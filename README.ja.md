## 概要

NeoDB Lite は [NeoDB](https://neodb.social) および互換インスタンス向けの非公式 Android クライアントです。スマートフォンで書籍、映画、ドラマ、音楽、ゲーム、ポッドキャスト、公演などを閲覧、検索、記録できます。

## 機能

### ブラウズ / 発見

- インスタンスログイン：NeoDB インスタンスのドメインを入力し、OAuth で認証します。
- 発見: カテゴリ別（書籍、映画、ドラマ、音楽、ゲーム、ポッドキャスト、公演）のトレンドを表示します。
- 検索：全カテゴリ横断検索、またはカテゴリを絞った検索とページネーションに対応。
- 詳細表示：カバー、タイトル、評価、説明、タグ、現在のアカウントの記録状態を表示。
- コミュニティ: 公開コメント、レビュー、ノートを表示し、Web 版へのリンクも提供。

### 記録管理

- 記録: 棚の状態（読みたい/読んでいる/読んだ/中断）、0〜10 の評価、ショートコメント、公開範囲を設定。記録の編集と削除も可能。
- マイ棚: 棚の状態ごとに記録を表示、カテゴリで絞り込み、カレンダーでの振り返りも可能。
- プロフィール: アカウント情報、棚の概要、最近完了したアイテム、各種設定を表示。

### 設定

- テーマ切り替え：複数のカラーテーマから選択可能。
- 言語切り替え：简体中文、繁體中文、日本語、English に対応。
- アプリアップデート：起動時に自動で新バージョンを確認（設定で無効化可能）。アプリ内ダウンロード、システムインストーラーの起動、ダウンロード再試行、手動ダウンロードに対応。

## 使い方

### インストール

[Releases](https://github.com/KrelinnBios/NeoDBLite/releases) から APK をダウンロードしてインストールしてください。

### システム要件

Android 7.0（API 24）以上。

### アップデート

アプリ起動時に GitHub Releases API を介して新バージョンを自動確認します（設定で自動確認を無効化可能）。設定画面から手動で確認することもできます。更新が見つかると、新バージョンとリリースノートを表示するダイアログが表示され、「手動ダウンロード」「後で」「ダウンロードしてインストール」の 3 つのオプションを提供します。

ダウンロード中はプログレスバーにパーセンテージとソース情報（GitHub 直リンクとミラーフォールバックに対応）を表示します。ダウンロード完了後、APK のバージョンと署名を検証してからシステムインストーラーを起動します。ダウンロードまたはインストールに失敗した場合はエラー内容を表示し、リトライできます。

## 技術情報

- 技術スタック：Kotlin、Jetpack Compose、Material 3、Retrofit、OkHttp。
- 認証方式：Mastodon 互換の OAuth 認可コードフローを使用して NeoDB インスタンスに接続。
- 更新仕組み：GitHub Releases API でバージョンチェック、ダウンロードした APK のバージョンと署名を検証。
