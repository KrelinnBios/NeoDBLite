## 概要

NeoDB Lite は [NeoDB](https://neodb.social) および互換インスタンス向けの非公式 Android クライアントです。スマートフォンで書籍、映画、ドラマ、音楽、ゲーム、ポッドキャスト、公演などを閲覧、検索、記録できます。

この README は、NeoDB Lite が現在提供している機能のみを説明します。

## 機能

- インスタンスログイン：NeoDB インスタンスのドメインを入力し、ブラウザ経由で認証します。
- 発見：書籍、映画、ドラマ、音楽、ゲーム、ポッドキャスト、公演のトレンドを閲覧できます。
- 検索：全カテゴリ横断またはカテゴリ指定で検索でき、最近の検索履歴も利用できます。
- 詳細表示：カバー、タイトル、評価、説明、タグ、外部リンク、現在の記録状態を表示します。
- 記録管理：棚の状態、評価、短評、タグ、公開範囲を設定し、既存の記録を編集または削除できます。
- マイ棚：状態、カテゴリ、タグ、タイトルキーワードで自分の記録を確認でき、カレンダー表示にも対応します。
- コレクション：自分のコレクションとその中のアイテムを閲覧できます。
- プロフィール：アバター、自己紹介、棚の統計、最近完了したアイテム、コレクション入口を表示します。
- 設定：テーマ、表示言語、更新確認、フィードバック、ログアウトに対応します。

## 使い方

### インストール

[Releases](https://github.com/KrelinnBios/NeoDBLite/releases) から APK をダウンロードしてインストールしてください。

### システム要件

Android 7.0（API 24）以上。

### アップデート

アプリ起動時または設定画面から新しいバージョンを確認できます。更新が見つかった場合は、アプリ内でダウンロードしてインストールするか、Releases ページから手動でダウンロードできます。

Android が上書きインストールを拒否する場合は、古いバージョンをアンインストールしてから Releases の APK を再度インストールしてください。

## データと安全性

- ログイン認証は、選択した NeoDB または互換インスタンスへアクセスするためにのみ使用されます。
- アイテム情報、カバー、評価、コメントは、利用中のインスタンスおよび関連ソースから取得されます。
- APK はこのリポジトリの Releases または信頼できる配布元からのみインストールしてください。

## 免責事項
