## 项目简介

NeoDB Lite 是面向 [NeoDB](https://neodb.social) 及兼容实例的非官方 Android 客户端，用于在手机上浏览、搜索和标记书影音游等条目。

以下说明仅描述 NeoDB Lite 当前实际提供的功能。

## 功能概览

- 实例登录：填写 NeoDB 实例域名后，通过浏览器授权登录。
- 发现浏览：按图书、电影、剧集、音乐、游戏、播客和演出查看趋势内容。
- 条目搜索：支持跨类目或限定类目搜索，保留最近搜索历史。
- 条目详情：展示封面、标题、评分、简介、标签、外部来源链接和当前账号的标记状态。
- 标记管理：可设置书架状态、评分、短评、标签、可见性，并修改或删除已有标记。
- 我的书架：按状态、类目、标签或标题关键词查看自己的标记，并提供日历视图。
- 收藏清单：查看自己的收藏单和清单内条目。
- 个人主页：展示头像、简介、书架统计、最近完成条目和收藏单入口。
- 辅助设置：支持多套主题、三种界面语言、更新检查、问题反馈和退出登录。

## 使用方式

### 安装使用

从 [Releases](https://github.com/KrelinnBios/NeoDBLite/releases) 下载 APK 后安装。

### 系统要求

Android 7.0（API 24）及以上。

### 更新方式

应用可在启动时或设置页检查新版本。检测到更新后，可以在应用内下载并安装，也可以前往 Releases 页面手动下载。

若系统提示无法覆盖安装，通常需要先卸载旧版，再从 Releases 重新安装。

## 数据与安全

- 登录授权只用于访问所选择的 NeoDB 或兼容实例。
- 条目数据、封面、评分和评论来自所登录实例及其关联来源。
- 请只从本仓库 Releases 或可信来源安装 APK，避免使用来源不明的改包版本。

## 内容边界

- 本项目为非官方客户端，与 NeoDB 项目及各实例运营方无隶属关系。
- 请遵守所登录实例的服务规则、内容规范和所在地法律法规。
